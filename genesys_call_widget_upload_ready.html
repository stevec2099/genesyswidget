Genesys Cloud Widget

Place a call

Upload-ready single-page widget that posts to /api/v2/conversations/calls when Place Call is selected. The JSON body uses the webpage Dialed Number as phoneNumber and My Number as callerId.

Connection

This version sends the API request directly from the browser, so it needs a valid Genesys Cloud OAuth access token.

payload = { phoneNumber: dialedNumber, callerId: myNumber, callerIdName }
